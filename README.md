## 介绍

多年后，后人提起今日：己亥年，庚子春，大疫，万人染疾，众惶恐，不出户，举国防疫，数月方休……

## 本地开发 & 阅读

本项目基于 vuepress 进行开发，以提供比 github mardown 更佳的阅读体验

依赖于 `node.js`、`yarn`、`vuepress` 等环境

```sh
# vuepress
yarn global add vuepress

# 本地开发
git clone https://github.com/gdut-yy/Leetcode-Hub.git
cd Leetcode-Hub/
yarn docs:dev
```
